# הוראות הטמעה — עמוד נחיתה "פשוט בריא" בוורדפרס + אלמנטור

---

## שיטה מומלצת: Full-Width Page עם HTML Block

### שלב 1 — יצירת עמוד חדש בוורדפרס

1. היכנסי לממשק הניהול: `sivanofiri.com/wp-admin`
2. לחצי על **Pages → Add New**
3. תני לעמוד שם, לדוגמה: `פשוט בריא - קורס`

### שלב 2 — הגדרת תבנית Full-Width

1. בצד ימין, תחת **Page Attributes** → **Template**, בחרי **Full Width** (או **Elementor Full Width** / **Canvas**)
2. זה מסיר את ה-Header וה-Footer של הת'מה — העמוד שלנו כולל ניווט ופוטר משלו

### שלב 3 — הוספת ה-HTML

1. לחצי על **Edit with Elementor**
2. גרור Widget מסוג **HTML** לתוך העמוד
3. פתחי את קובץ `index.html` בעורך טקסט (Notepad, VS Code וכו')
4. סמני הכל (Ctrl+A) → העתיקי (Ctrl+C)
5. הדביקי את כל הקוד לתוך שדה ה-HTML באלמנטור
6. לחצי **Update**

### שלב 4 — עדכון לינק התשלום

בקוד, חפשי את המחרוזת `#PAYMENT_LINK` (מופיעה 2 פעמים) והחליפי בלינק האמיתי לדף התשלום.

## טיפים חשובים

- **מובייל**: העמוד מותאם מלאה למובייל — בדקי ב-Chrome DevTools (F12 → Toggle Device)
- **מהירות**: התמונות מוטמעות ישירות בקוד (base64) — אין תלות בשרת חיצוני
- **לינק CTA**: יש 3 כפתורים בעמוד — כולם מצביעים ל-`#PAYMENT_LINK` — עדכני את כולם
- **פונט Heebo**: נטען מ-Google Fonts — דורש חיבור לאינטרנט
